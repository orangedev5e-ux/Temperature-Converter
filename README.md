# Temperature Converter Application

This is a fully responsive web application that converts temperatures between Celsius, Fahrenheit, and Kelvin in real-time. 

## 🚀 Challenge Source
This project was built as a solution to the [Roadmap.sh Temperature Converter Challenge](https://roadmap.sh/projects/temperature-converter).
https://roadmap.sh/projects/temperature-converter
## 🛠️ Features
- Custom responsive UI designed for Laptop, Tablet, and Mobile displays.
- Strict JavaScript type conversion and condition matrix routing.
- Two-decimal output precision.
